============================
Timesheet Automatic Reminder
============================

This module extends the functionality of time sheet to support remind employee
who didn't fill their TMS by automatic sending emails to each other at morning
and CC their managers at noon every day(Will Check the TMS 7 days before).

Installation
============

To install this module, you need to:

#. Have basic modules installed (hr_timesheet, project_timesheet, project_issue_sheet)
