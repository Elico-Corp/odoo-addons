=======================
Report Py3o Multisheet
=======================

This technical module modifies the template required by 'report_py3o' module.
It adds multi-sheet management into the template. Each sheet repeats the same header and footer,
part of first sheet. The body can be customized with specified list.

Installation
============
To install this module, you need to install ezodf python library:

.. code-block::

  sudo pip install ezodf
